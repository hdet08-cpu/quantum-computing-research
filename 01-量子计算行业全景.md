# 量子计算行业全景 · 2026年中

## 一、六大技术路线对比

### 1. 中性原子 Neutral Atom ⭐ 最快进展

用激光把单个原子"夹"在空中排成阵列，通过里德堡态实现量子门操作。

| 指标 | 最佳数据 | 来源 |
|---|---|---|
| 最大比特阵列 | 6,100 原子 | Caltech Endres 组 (2025.9) |
| 最多逻辑比特 | 96 个 | QuEra/Harvard/MIT (2025.1) |
| 最高双比特门保真度 | 99.94% (损失后选择) | Harvard/QuEra (2026.4) |
| 最大无缺陷寄存器 | 1,024 原子 | Pasqal (2026.4) |
| 最长连续运行 | 2+ 小时 (3000 原子) | Harvard/MIT/QuEra (2025.9) |
| 最长相干时间（阵列内） | 13 秒 | Caltech (2025.9) |

**主要公司：**
- **QuEra** 🇺🇸 — 96 逻辑比特领先全平台，$2.8亿融资，Harvard/MIT spinout
- **Pasqal** 🇫🇷 — 商业化最快（7台QPU已交付），$5.5亿+融资，$20亿估值，拟上市
- **Atom Computing** 🇺🇸 — 锶原子 + 微软合作，24 逻辑比特，CHIPS 法案 $1亿
- **Infleqtion** 🇺🇸 — 已上市 NYSE，全栈量子（计算+传感+时钟），12 逻辑比特
- **planqc** 🇩🇪 — 光晶格路线（非光镊），锶原子，Max Planck spinoff

## 四、2025-2026 行业趋势

1. **容错时代开启**：多平台证明低于阈值的纠错，中性原子逻辑比特数领先
2. **资本大规模涌入**：2025年近 $100亿，多家公司 SPAC 上市（Xanadu $31亿、Pasqal $20亿、Infleqtion $5.5亿）
3. **并购整合**：IonQ 完成 $31.5亿+ 并购，D-Wave 收购 QCI
4. **政府战略投资加速**：美国 CHIPS 法案、中国十五五规划、欧洲量子旗舰
5. **编码效率成为新战场**：Quantinuum 2:1 vs 传统 surface code 100:1，qLDPC 码（IBM/IonQ）~12:1
6. **"量子优势"竞赛**：多家公司（Pasqal、QuEra、IBM）目标 2026 年底前在实际问题上展示量子优势
