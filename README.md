## 关键结论速览

### 行业格局
- 中性原子在 2025-2026 年从"潜力路线"跃升为**容错量子计算的领跑者**
- QuEra（Harvard/MIT）以 **96 个逻辑比特**领先所有平台
- 2025 年全球量子计算行业融资近 **$100 亿**

### 中性原子核心优势
- 可扩展性最强（6100 比特已演示）
- 可重构连通性（光镊物理移动原子）
- 纠错友好（擦除错误 + 横向门）

### 原子矩阵（MatriQ）
- 国内中性原子先发者，2048 无缺陷比特阵列
- **待验证**：保真度数据未公开、逻辑比特=0、与中科大团队关系不透明

### EdenCode
- AI-native 量子纠错解码器，方向正确
- **核心风险**：延迟 1000x gap（ms→μs）、独立解码器公司生态位存疑、CTO 未全职
